# Bront

Bront is a fork of Ubuntu Mono geared towards programming.

## How does Bront differ from Ubuntu Mono?

In Bront:

- `$` has a full pillar
- `0` has a slash
- `_` is narrower
- `-` is wider (same width as `_`)
- `*` is centered vertically
- `G` has a horizontal bar
- `%` has actual circles
